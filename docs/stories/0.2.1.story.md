## 📋 Technical Details

### Email Service Options

**Option A: SendGrid (Recommended for production)**
```
- Cost: Pay-as-you-go
- Setup: Create SendGrid account, get API key
- Env: SENDGRID_API_KEY=xxx, EMAIL_SERVICE=sendgrid
- Pros: Reliable, good analytics, webhook support
- Cons: Requires external service
```

**Option B: Supabase Email (Built-in)**
```
- Cost: Included in Supabase plan
- Setup: Enable in Supabase dashboard
- Env: SUPABASE_EMAIL_SERVICE=true
- Pros: Already using Supabase, no extra service
- Cons: Limited customization
```

### Email Service Interface

```typescript
// src/lib/emailService.ts
interface EmailService {
  send(options: {
    to: string | string[];
    subject: string;
    template?: string;
    variables?: Record<string, unknown>;
    text?: string;
    html?: string;
    from?: string;
  }): Promise<{ success: boolean; messageId?: string; error?: string }>;

  sendTemplate(templateName: string, data: {
    to: string[];
    variables: Record<string, unknown>;
  }): Promise<{ success: boolean; sent: number; failed: number; errors: string[] }>;
}
```
